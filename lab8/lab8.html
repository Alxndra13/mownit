Wstęga Newtona¶

In [ ]:
f(x) = x^3 - 1

# obliczenie pochodnej f i zbudowanie mapowania iteracyjnego dla metody Newtona, zwraca mapowanie jako wynik
function get_map(f)
    @variables x
    Dx = Differential(x)
    map = x - f(x) / Dx(f(x)) |> expand_derivatives
    update_expr = build_function(map, x)
    return eval(update_expr)
end

update = get_map(f)

# iteracyjna aktualizacja punktu x za pomoc funkcji update
function applyiteratively(x, update; n=100)
    for i in 1:n
        x = update(x)
    end
    return x
end

# granice obszaru i krok
lower = -2 - 2im
upper = 2 + 2im
step = 0.5e-2

# macierz Z0, ktra zawiera punkty zespolone na obszarze okrelonym przez lower, upper i step, macierz ta suy jako pocztkowe punkty iteracji metody Newtona
Z0 = [a+b*im for b in real(lower):step:real(upper),
                    a in imag(lower):step:imag(upper)]

Z100 = applyiteratively.(Z0, update)

# wykres typu heatmap, ktry przedstawia kty liczby zespolonej w macierzy Z100, kty s kodowane przez kolory
heatmap(angle.(Z100), colorbar=false, color=:rainbow, ticks=false)

W powyszym kodzie generowana jest wstga Newtona na podstawie zastosowania metody Newtona do znalezienia pierwiastkw funkcji f(x) = x^3 - 1.

Proces generowania wstgi Newtona polega na iteracyjnym aktualizowaniu punktw w przestrzeni zespolonej za pomoc mapowania iteracyjnego, ktre jest zdefiniowane w funkcji get_map.

Kady punkt z macierzy Z0 suy jako pocztkowy punkt iteracji. Nastpnie, dla kadego punktu, iteracyjnie stosowana jest funkcja update, ktra odnajduje kolejne przyblienia pierwiastka funkcji f(x). Liczba iteracji jest kontrolowana przez parametr n w funkcji applyiteratively.

Wartoci punktw po iteracjach s zapisywane w macierzy Z100. Wykres typu heatmap jest generowany na podstawie ktw liczby zespolonej w macierzy Z100. Kty te s kodowane przez kolory przy uyciu palety barw rainbow.

Zwizek z metod Newtona polega na iteracyjnym przyblianiu pierwiastkw funkcji f(x) poprzez stosowanie mapowania iteracyjnego, ktre jest zdefiniowane jako rnica midzy aktualnym punktem a wartoci funkcji f(x) podzielon przez pochodn funkcji f(x). Metoda Newtona polega na powtarzaniu tego procesu a do zblienia si do rzeczywistego pierwiastka funkcji. W przypadku wstgi Newtona, iteracje s przeprowadzane na siatce punktw w przestrzeni zespolonej, a kty liczby zespolonej reprezentuj stopie zbienoci do pierwiastkw.
