Zadanie 1 (2 pkt)¶

Ustal losowe (referencyjne) x rozmiaru 1000 oraz losowe A rozmiaru 1000x1000, policz b=A*x. Nastepnie rozwi rwnanie Ax=b trzema metodami:

  • inv()
  • \
  • factorize()

Porwnaj jako wyniku (zmierzon jako dugo rnicy wektorw wyniku oraz referencyjnego x) oraz czas wykonania (@time). UWAGA: pierwsze wykonanie funkcji w Julii zawiera czas kompilacji tej funkcji, dlatego czas mierzymy od drugiego wywoania !

result = DataFrame(Method=String[], Error=Float64[], Time=Float64[])

for i in 1:10

    # Generowanie losowych danych
    x = rand(1000)
    A = rand(1000, 1000)
    b = A * x

    inv_diff = x - inv(A)*b
    inv_res = sqrt(dot(inv_diff,inv_diff))
    inv_time = @elapsed inv(A)*b

    norm_diff = x - A\b
    norm_res = sqrt(dot(norm_diff, norm_diff))
    norm_time = @elapsed A\b

    fact_diff = x - factorize(A)\b
    fact_res = sqrt(dot(fact_diff, fact_diff))
    fact_time = @elapsed factorize(A)\b

    if i == 1
        continue
    end

    push!(result, ("inv", inv_res, inv_time))
    push!(result, ("\\", norm_res, norm_time))
    push!(result, ("factorize", fact_res, fact_time))
end

grouped_result = combine(groupby(result, :Method), :Error => mean, :Time => mean)

Metoda pseudoinwersji macierzy¶

function print_matrix(matrix)
    for i in 1:size(matrix, 1)
        for j in 1:size(matrix, 2)
            print(matrix[i, j], " ")
        end
        println()
    end
end

print_matrix(A)
println("wyznacznik: ", 0)

1 2 3 
4 5 6

Dla danej macierzy A, jej pseudoinwersja (oznaczana jako A) spenia nastpujce warunki:

  • Jeeli macierz A jest odwracalna, to pseudoinwersja A jest rwna odwrotnoci A, czyli A = A.
  • Jeeli macierz A nie jest odwracalna, pseudoinwersja A pozwala nam znale rozwizanie x rwnania Ax = b, ktre minimalizuje norm euklidesow ||Ax - b||. Oznacza to, e A jest "najlepszym" przyblieniem odwrotnoci A w sensie minimalizacji bdu.

Faktoryzacja QR rozkada macierz A na iloczyn macierzy ortogonalnej Q i macierzy trjktnej grnej R.

Pseudoinwersja macierzy A jest obliczana w nastpujcy sposb:

  • Obliczenie faktoryzacji QR macierzy A: A = QR.
  • Obliczenie odwrotno macierzy R: R.
  • Obliczenie pseudoinwersji macierzy A: A = R * Q'.

Pseudoinwersja macierzy A jest wynikiem mnoenia macierzy odwrotnej R przez transpozycj macierzy Q. Ta procedura wykorzystuje wasnoci faktoryzacji QR i pozwala na znalezienie pseudoinwersji macierzy A.

W praktyce, uywanie pseudoinwersji macierzy jest przydatne w przypadkach, gdy rozwizanie liniowego rwnania Ax = b jest niemoliwe lub numerycznie niestabilne. Pseudoinwersja macierzy pozwala na znalezienie optymalnego przyblienia rozwizania i moe by stosowana w wielu problemach, takich jak dopasowywanie krzywych, regresja nieliniowa, rozwizywanie ukadw rwna nadokrelonych i wiele innych.

function compute_pseudoinverse(A)
    Q, R = qr(A)
    R_inv = inv(R)
    pseudoinverse = R_inv * Q'
    return pseudoinverse
end

# Obliczenie pseudoinversji macierzy
pseudoinverse = compute_pseudoinverse(A)

println("Pseudoinwersja macierzy A:")
print_matrix(pseudoinverse)
